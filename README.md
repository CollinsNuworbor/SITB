# SITB - Solving Is The Best

A feature-rich calculator web application built with HTML, CSS, and JavaScript.

## Features

- **Basic Operations**: Addition, subtraction, multiplication, division, modulo
- **Advanced Functions**: Square root, trigonometric functions (sin, cos, tan)
- **Scientific Functions**: Logarithm, powers (^), factorial, constants (π, e)
- **History Tracking**: All calculations are saved and displayed in the history panel
- **User-Friendly Interface**: Clean, dark-themed design with responsive buttons
- **Error Handling**: Invalid inputs are caught and handled gracefully

## How to Use

1. Open `index.html` in your web browser
2. Click buttons to enter numbers and operations
3. Press `=` to calculate the result
4. View your calculation history on the right side
5. Use `C` to clear the display or `CE` to delete the last character

## Files

- `index.html` - Calculator interface and structure
- `style.css` - Styling and layout
- `script.js` - Calculator logic and functionality

## Functions

- **Basic**: C (Clear), CE (Backspace), +, −, ×, ÷, %, ^
- **Scientific**: √ (Square Root), sin, cos, tan, log, π, e, n! (Factorial)
